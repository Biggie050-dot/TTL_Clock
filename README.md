# TTL Clock Project

## Overview

This project is based on the TTL Clock designed by Sergey Kiselev:
https://github.com/skiselev/ttl-clock

The purpose of this project is to redesign, analyze and reproduce a TTL-based digital clock using discrete 74xx logic integrated circuits.

This project was developed for the course **Project Ontwerpen 2025-2026** in Electronics-ICT.

The project includes:

* Electronic schematic design
* PCB development
* Component selection and analysis
* Cost estimation
* Assembly and testing
* 3D enclosure design
* Technical documentation
